## How to use?
1. Clone this repository to hosting directory (e.g.. htdocs folder in XAMPP)
2. Start the web server
3. Open web browser then go to this URL: http://localhost/CITreasury-main/initialize.php
4. This will initialize the database and show relevant information about the admin account
5. Alternative way was resetting the database: http://localhost/CITreasury-main/reset.php
6. This will re-initialize the database and show relevant information about the admin account
